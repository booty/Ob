OtakuBooty's API

This page is a demonstration. It's not the "new OtakuBooty" — it's the plumbing of the new OB.

What the fuck is an API?

It lets nerds build shit using data and shit from OB. For example, here's a list of the forums. It's a blob of data (in this case, a list of forums) that somebody else's code can use. Somebody could use this to display a list of OB's forums somewhere.

Discussion / Support

For now, best place to discuss is the thread on OB. Let everybody know your issues/triumphs.

Who the fuck is this page for?

  • It's for coders. So you can see examples of the API in action, see the API endpoint URLs and maybe steal some code.
  • It's for the people writing the API. So we can test stuff.
  • It's for everybody else. So you can peek over our shoulders and see us making stuff.

I'm A Coder! I Make Shit!

Then you want the documentation, friend! Also, consider stealing examples from this page and the associated Javascript file, obApi.js.

I Want To Be a Coder

To make web pages with this stuff, you just need a basic knowledge of HTML, Javascript, and probably jQuery (in that order). No fancy software. Pros use text editors like Notepad.exe, but beefier. Notepad++ is a good free one to start with.
