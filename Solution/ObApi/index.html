OtakuBooty's API

This page is a demonstration. It's not the "new OtakuBooty" or anything.

What the fuck is an API?

It lets nerds build shit using data and shit from OB. For example, here's a list of the forums. It's a blob of data (in this case, a list of forums) that somebody else's code can use. Somebody could use this to display a list of OB's forums somewhere.

Who the fuck is this page for?

  • It's for coders. So you can see examples of the API in action, see the API endpoint URLs and maybe steal some code.
  • It's for the people writing the API. So we can test stuff.
  • It's for everybody else. So you can peek over our shoulders and see us making stuff.
